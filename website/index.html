Frequently Asked Questions

Everything you need to know about Moonforge

Is Moonforge free to use?

Yes! Moonforge is open-source and free to use. The project is licensed under the MIT License, which means you can use it for personal or commercial purposes.

Which platforms are supported?

Moonforge currently supports Windows and Linux desktops, web browsers, and Android. macOS and iOS support are actively in development.

Do I need an internet connection?

Moonforge works offline! The app uses an offline-first architecture, so you can work on your campaigns anywhere. Changes sync automatically when you're back online.

Can I share campaigns with my players?

Yes! Moonforge includes a Player View feature that allows read-only access to campaign content. Your players can explore the world without seeing your DM secrets.

Is my data secure?

Your data is stored securely using Firebase with automatic backups. The app uses industry-standard encryption and authentication to protect your campaigns.

Can I import existing campaigns?

Import tools are on the roadmap for Phase 3. Currently, you can manually create your campaigns within Moonforge using the rich text editor and entity management tools.

Does it work with D&D Beyond?

Direct integration with D&D Beyond is not currently available, but we're exploring partnerships and import options for future releases.

How can I contribute?

We welcome contributions! Check out our GitHub repository for contribution guidelines, open issues, and development documentation.
